WooCommerce 2.1+ payment gateway for Tranzila

== Description ==

This plugin provides a gateway from WooCommerce to Tranzila.

It works ‘out of the box’ from WordPress but requires special setting in Tranzila’s admin

Read the Installation instructions!!!!
 
Disclaimer: Use this at your own risk, I accept no responsibility whatsoever for any losses resulting from the use of this plugin.

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload the `woocommerce-tranzila` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Login to https://direct.tranzila.com/merchant
4. Set `success URL` to YOUR_SITE_URL/checkout/order-received/?tranzila=successful-payment
